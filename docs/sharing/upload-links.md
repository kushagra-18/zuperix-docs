# Magic Upload Links 🪄

Stop chasing attachments in emails. Zuperix **Public Upload Links** give you a secure, frictionless way to collect assets from clients, vendors, or guest contributors without requiring them to create an account.

It’s not just an upload button; it’s a high-performance ingestion engine that organizes your library before the files even arrive.

## Why it’s better than anything else

Our Public Upload Links are engineered for scale and speed. Whether your contributor is sending a 5KB logo or a 50GB RAW video file, Zuperix handles it with ease.

### 🚀 High-Speed Ingestion 
We use the same enterprise-grade upload infrastructure for guests as we do for registered users. 
- **Direct-to-Cloud**: Files go straight to secure storage, bypassing the web server for maximum speed.
- **Multipart Acceleration**: Large files are automatically split into chunks and uploaded in parallel, ensuring reliability even on spotty connections.

### 🤖 Automagic Organization
The messiest part of guest uploads is the lack of metadata. Not with Zuperix.
When you generate a link, you can pre-attach:
- **Target Categories**: Automatically sort files into specific folders.
- **Smart Tags**: Apply project tags (e.g., `Campaign-2026`, `Vendor-X`) instantly.
- **Deep Metadata**: Pre-fill complex metadata fields. When the files arrive, they are already indexed and fully searchable.

### 🛡️ Ironclad Security & Control
You are in the driver's seat. Every link you generate comes with granular restrictions:
- **Expiration Dates**: Set it and forget it—links deactivate automatically.
- **Upload Counts**: Limit the link to `X` number of files to prevent spam.
- **File Restrictions**: Only allow specific types (Images, PDFs, Videos) or set maximum file size limits.
- **Manual Kill-Switch**: Disable or delete any link instantly from your dashboard.

### 👤 Seamless Attribution
Never wonder "where did this come from?" again. 
Every asset uploaded via a link is automatically tagged with a **Creator Attribution**. The system remembers *who* generated the link and marks the assets as **"Uploaded by [You]"**, providing a clear audit trail for guest contributions.

## How to use it

:::tip Direct Dashboard Upload
For internal team members, you can always upload files directly via the **Upload** button in the dashboard header.
:::

1. **Generate**: Click the **Upload** button in the Assets view and select **Generate Upload Link**.
2. **Configure**: Set your presets—tags, category, and metadata.
3. **Control**: Define your expiry and limits.
4. **Share**: Copy the magic link and send it.

Your contributors get a beautiful, clean interface that works perfectly on desktop and mobile. All they have to do is drop their files and watch the progress bar fly.
